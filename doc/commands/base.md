# Base functions

## Evaluate (EVAL)

Evaluate the object at stack level 1.

Mapped to the _ R/S _ key

`X` ▶ Result of `X` evaluation

## Negate (NEG)

Negate the value in level 1.

Mapped to the _ +/- _ key

`X` ▶ `0-X`

## Invert (INV)

Invert the value in level 1

Mapped to the _ 1/X _ key

`X` ▶ `1/X`
